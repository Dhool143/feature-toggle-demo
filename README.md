# Feature Toggle Demo

This project demonstrates conditional rendering in React using a reusable FeatureToggle component.

## Features
- React functional component
- Props used: isEnabled and featureName
- Conditional rendering based on props
- Multiple feature examples

## How to Run

1. Install dependencies
npm install

2. Start development server
npm run dev

3. Open browser
http://localhost:5173
